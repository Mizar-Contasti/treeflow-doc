# 📜 Historial de Cambios

> Para una versión más detallada del historial de cambios con todas las características y mejoras, consulta el [Historial de Cambios Detallado](/docs/Misc/HistorialDetallado).


## 🎉 Versión Actual

<details open>
<summary><strong>V-0.4 (10/2025)</strong></summary>

### 🤖 Reforestación Mejorada (IA Generativa)
> -  **Pipeline de 6 Fases**: Proceso más granular y preciso para generación de contenido
>    - Fase 1: Abonar (Expansión Conversacional)
>    - Fase 2: Análisis Conversacional
>    - Fase 3: Estructura Básica
>    - Fase 4: Enriquecimiento con Sinónimos
>    - Fase 5: Estructuración Semántica
>    - Fase 6: Aplicación al Bot
> -  **Formato Estándar**: Consistencia en todas las fases con `generated_text` + metadata
> -  **Modelos Actualizados**: Groq con modelos activos (llama-3.1-8b-instant, mixtral-8x7b-32768, etc.)
> -  **Frameworks Modulares**: Sistema extensible de generación (no_framework, langgraph)
> -  **Contexto Inteligente**: Considera intents y entities existentes del bot antes de generar
> -  **Funcionamiento Interno**: Procesamiento por fases con validación y logging detallado

### 🎨 Experiencia de Usuario
> -  **Modo Oscuro**: Tema dark mode completo en toda la aplicación
> -  **Dashboard Mejorado**: Vista general con métricas y accesos rápidos
> -  **UI con Tabs**: Navegación por pestañas para mejor organización
> -  **Sistema de Caché**: Navegación sin parpadeos con caché inteligente de 60 segundos
> -  **Precarga Automática**: Datos de workspace y usuarios disponibles instantáneamente
> -  **Invalidación Selectiva**: Refresco inteligente solo de datos modificados
> -  **Permisos Granulares**: Control de acceso por rol (admin, desarrollador, viewer)
> -  **Navegación Fluida**: Sin necesidad de recargar página (F5)

### 🌿 Entidades del Sistema (Hojas)
> -  **@sys.any**: Captura cualquier texto
> -  **@sys.number**: Números enteros y decimales
> -  **@sys.date**: Fechas en múltiples formatos
> -  **@sys.time**: Horas y tiempos
> -  **@sys.email**: Direcciones de correo electrónico
> -  **@sys.phone**: Números telefónicos
> -  **@sys.url**: URLs y enlaces web
> -  **@sys.currency**: Monedas y cantidades monetarias
> -  **@sys.percentage**: Porcentajes
> -  **@sys.ordinal**: Números ordinales (primero, segundo, etc.)

### 🧪 Fertilizantes (Webhooks)
> -  **Webhooks Avanzados**: Sistema completo de webhooks con testing en tiempo real
> -  **Tools Integration**: Herramientas integradas para debugging y monitoreo
> -  **Testing en Vivo**: Prueba webhooks directamente desde la interfaz
> -  **Logs Detallados**: Registro completo de requests y responses
> -  **Retry Logic**: Reintentos automáticos en caso de fallo

### 🔌 Integraciones
> -  **Widget Web (QWidget)**: Integración embebible en 2 líneas de código
> -  **CDN Global**: Disponible vía jsdelivr para uso inmediato
> -  **Web Component**: Shadow DOM con estilos aislados
> -  **API Nativa**: Endpoint `/message` compatible con TreeFlow

### 🛠️ Desarrollo
> -  **LocalTunnel**: Exposición de desarrollo local a internet con URLs públicas
> -  **Bypass Automático**: Headers para evitar páginas de autenticación
> -  **Docker Optimizado**: Configuración mejorada para desarrollo

</details>
