# Telefonía

Integra tu árbol conversacional con sistemas de telefonía para crear experiencias de voz interactivas y automatizar llamadas telefónicas.

## Características

- **Llamadas entrantes** automatizadas
- **Llamadas salientes** programadas
- **Reconocimiento de voz** (Speech-to-Text)
- **Síntesis de voz** (Text-to-Speech)
- **Transferencia a agentes** humanos
- **IVR avanzado** (Interactive Voice Response)

## Configuración

La configuración de Telefonía se realiza desde el panel de Injertos en TreeFlow.
