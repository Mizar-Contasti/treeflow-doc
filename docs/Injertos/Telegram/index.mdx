# Telegram

Conecta tu árbol conversacional con Telegram para llegar a millones de usuarios en una de las plataformas de mensajería más populares del mundo.

## Características

- **Mensajes de texto** con formato enriquecido
- **Botones inline** para respuestas rápidas
- **Comandos personalizados** (/start, /help, etc.)
- **Multimedia** (imágenes, videos, documentos)
- **Grupos y canales** para comunicación masiva

## Configuración

La configuración de Telegram se realiza desde el panel de Injertos en TreeFlow.
