# Facebook

Conecta tu árbol conversacional con Facebook Messenger para interactuar con usuarios en una de las plataformas sociales más grandes del mundo.

## Características

- **Messenger Bot** automatizado
- **Botones de respuesta rápida** y menús persistentes
- **Plantillas enriquecidas** (carruseles, listas, recibos)
- **Multimedia** (imágenes, videos, GIFs)
- **Integración con páginas** de Facebook
- **Webviews** para experiencias extendidas

## Configuración

La configuración de Facebook se realiza desde el panel de Injertos en TreeFlow.
