# SMS

Conecta tu árbol conversacional con servicios de SMS para llegar a usuarios a través de mensajes de texto tradicionales.

## Características

- **Mensajes de texto** bidireccionales
- **Respuestas automáticas** basadas en palabras clave
- **Notificaciones** y alertas programadas
- **Soporte internacional** para múltiples países
- **Integración con operadoras** locales

## Configuración

La configuración de SMS se realiza desde el panel de Injertos en TreeFlow.
