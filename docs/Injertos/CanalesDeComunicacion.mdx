# Canales de Comunicación

Los **Canales de Comunicación** son las diferentes plataformas y medios a través de los cuales tus usuarios pueden interactuar con tus chatbots de TreeFlow.

TreeFlow utiliza el concepto de **Injertos** para gestionar las integraciones con diferentes canales de comunicación.

## Canales Disponibles

### 📱 Canales de Mensajería
- WhatsApp Business API
- Facebook Messenger  
- Telegram
- Instagram Direct
- Twitter DM

### 🌐 Canales Web
- Widget Web embebible
- Chat Iframe

### 🎙️ Canales de Voz
- Telefonía IP
- Google Assistant
- Amazon Alexa

### 💼 Canales Empresariales
- Microsoft Teams
- Slack
- Discord

---

:::tip Configuración de Injertos

Para configurar cualquiera de estos canales, navega a la sección de **Injertos** en tu workspace de TreeFlow y selecciona el canal que deseas integrar.

:::
