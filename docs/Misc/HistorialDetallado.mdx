## Versiones Lanzadas

<details open>


<summary>

### <strong>Versión 0.3 (08/2025) - Versión Actual</strong>

</summary>

#### Inteligencia Artificial Generativa

> - ✅ **Sistema de Reforestación**
>   - Generación automática de Hojas (entidades) usando IA
>   - Creación automática de Intenciones (intents) con patrones naturales
>   - Motor generativo universal adaptable a cualquier dominio o industria
>   - Integración con modelo Groq llama-3.3-70b-versatile
>   - Análisis contextual del Árbol existente antes de generar contenido
>
> - ✅ **Estrategia de Entidades Inteligente**
>   - Separación automática entre entidades de acciones (verbos) y objetos (sustantivos)
>   - Generación de sinónimos inteligentes para cada entidad
>   - Nombres de entidades naturales sin prefijos artificiales
>   - Cobertura completa de variaciones lingüísticas
>
> - ✅ **Variables Dinámicas en Respuestas**
>   - Formato `{variable}` para personalización automática
>   - Variables que coinciden con nombres de entidades capturadas
>   - Respuestas contextuales adaptadas al dominio específico
>   - Sustitución automática en tiempo real durante conversaciones
>
> - ✅ **Flujo de Previsualización Completo**
>   - Vista previa de todos los cambios antes de aplicar
>   - Revisión detallada de entidades y patrones generados
>   - Opción de cancelar o aplicar modificaciones
>   - Validación automática de estructura TreeFlow
>
> - ✅ **Historial de Sesiones Generativas**
>   - Registro completo de todos los prompts utilizados
>   - Tracking de cambios generados y aplicados
>   - Información de fecha, hora y usuario para cada sesión
>   - Posibilidad de revisar y aprender de sesiones anteriores

#### Arquitectura y Mejoras Técnicas

> - ✅ **Backend Generativo Independiente**
>   - Servicio separado ejecutándose en puerto 8001
>   - Arquitectura de microservicios para escalabilidad
>   - Comunicación API entre backend principal y generativo
>   - Aislamiento de dependencias de IA del sistema principal
>
> - ✅ **Integración con Groq**
>   - Modelo llama-3.3-70b-versatile para generación de alta calidad
>   - Configuración optimizada de temperatura y parámetros
>   - Manejo robusto de errores y timeouts
>   - Respuestas en formato JSON estructurado
>
> - ✅ **Endpoints de Sobreescritura Simplificados**
>   - API optimizada para aplicación masiva de cambios
>   - Endpoints `/entities/{tree_id}/overwrite` y `/intents/{tree_id}/overwrite`
>   - Validación automática contra modelos Pydantic
>   - Transacciones atómicas para consistencia de datos
>
> - ✅ **Validación Automática Completa**
>   - Estructura EntityDef completa con todos los campos requeridos
>   - Generación automática de UUIDs únicos para todas las entidades
>   - Validación de compatibilidad con modelo Pydantic existente
>   - Normalización automática de estructura de datos
>
> - ✅ **Logging y Trazabilidad Exhaustiva**
>   - Registro detallado de todo el flujo generativo
>   - Información de debugging para resolución de problemas
>   - Métricas de rendimiento y tiempo de respuesta
>   - Trazabilidad completa desde prompt hasta aplicación
>
> - ✅ **Docker Compose Integrado**
>   - Despliegue unificado de todos los servicios
>   - Configuración de red interna entre contenedores
>   - Variables de entorno centralizadas
>   - Escalabilidad horizontal preparada

</details>

### <strong>Versión 0.2 (07/2025)</strong>

</summary>

#### Sistema de Workspaces
> - ✅ **Cuentas Independientes**
>   - Sistema completo de workspaces para separación de proyectos
>   - Aislamiento total de datos entre workspaces
>   - Cada workspace mantiene sus árboles, configuraciones y datos independientes
>   - Gestión de múltiples proyectos desde una sola cuenta
>
> - ✅ **Gestión Multi-Workspace**
>   - Creación y administración de espacios de trabajo
>   - Selector intuitivo para cambiar entre workspaces
>   - Configuraciones independientes por workspace
>   - Organización por cliente, proyecto o departamento
>
> - ✅ **Interfaz de Gestión**
>   - Panel de control centralizado
>   - Navegación fluida entre workspaces
>   - Indicadores visuales del workspace activo
>   - Gestión de configuraciones por workspace

#### Sistema de Fertilizantes (Webhooks Avanzados)
> - ✅ **Fertilizante Principal**
>   - Webhook único por workspace para procesamiento global
>   - Toggle de activación/desactivación
>   - Configuración de URL, autenticación Basic Auth y timeout
>   - Procesamiento automático de todas las respuestas del árbol
>
> - ✅ **Fertilizantes Secundarios**
>   - Múltiples webhooks especializados por integración
>   - Configuración individual con nombre y descripción
>   - Soporte completo para métodos HTTP (GET, POST, PUT, PATCH, DELETE)
>   - Timeouts configurables por fertilizante
>
> - ✅ **Testing Individual en Tiempo Real**
>   - Modal de pruebas especializado para cada fertilizante
>   - Peticiones HTTP reales a endpoints configurados
>   - Visualización completa de request y response
>   - Botón de test individual por fertilizante secundario

#### Sistema de Injertos (Integraciones)
> - ✅ **Injerto Web - Widget Embebible**
>   - Widget de chat completamente funcional para sitios web
>   - Integración súper simple en 2 líneas de código
>   - Web Component personalizado con Shadow DOM
>   - Disponible globalmente vía CDN (jsDelivr)
>
> - ✅ **Personalización Avanzada**
>   - Sistema completo de variables CSS (custom properties)
>   - Configuración de colores, iconos y comportamiento
>   - Personalización visual sin modificar código
>   - Soporte para temas personalizados
