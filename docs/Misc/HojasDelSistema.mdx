# Hojas del Sistema

import FutureFeatureNotice from '@site/src/components/FutureFeatureNotice.mdx';

<FutureFeatureNotice version="v0.2" />

Las Hojas del Sistema en TreeFlow son entidades predefinidas que están optimizadas para reconocer y extraer tipos de datos comunes sin necesidad de configuración adicional. Estas hojas facilitan la creación de experiencias conversacionales robustas al proporcionar extractores de información ya entrenados y listos para usar.

![Hojas del Sistema](/img/docs/hojas_sistema.svg)

## Lista Completa de Hojas del Sistema

| Hoja | Descripción | Ejemplos de formato |
|------|-------------|--------------------| 
| **sys.any** | Acepta cualquier texto como entrada válida. | "me gusta", "quiero información", "cualquier texto que el usuario escriba" |
| **sys.number** | Detecta números. | "42", "tres", "3.14", "mil doscientos" |
| **sys.currency** | Detecta montos monetarios. | "$100", "50 euros", "1,200.50 pesos" |
| **sys.date** | Detecta fechas en diversos formatos. | "15 de marzo", "15/03/2025", "mañana", "próximo lunes" |
| **sys.time** | Detecta horas en diversos formatos. | "3 PM", "15:30", "mediodía" |
| **sys.email** | Detecta direcciones de correo electrónico. | "usuario@ejemplo.com", "nombre.apellido@dominio.co.uk" |
| **sys.phone** | Detecta números de teléfono. | "+52 123 456 7890", "4421234567" |
| **sys.url** | Detecta URLs. | "https://ejemplo.com", "www.ejemplo.com" |
| **sys.percentage** | Detecta porcentajes. | "25%", "25 por ciento" |

## Uso Recomendado

Las Hojas del Sistema están diseñadas para facilitar la extracción de información común sin necesidad de crear hojas personalizadas. Para utilizarlas:

1. Incluye la referencia a la hoja del sistema en tus patrones de rama usando la sintaxis `@sys.nombre_hoja`
2. TreeFlow automáticamente extraerá y normalizará los valores que coincidan con el formato esperado

### Ejemplo en un Patrón de Rama

```
Quiero reservar una mesa para el @sys.date a las @sys.time
Necesito información sobre mi pedido #@sys.number
Envíame los detalles a @sys.email
```

## Ventajas de las Hojas del Sistema

- **Robustez**: Están optimizadas para reconocer múltiples variaciones y formatos regionales
- **Normalización**: Convierten automáticamente los valores extraídos a formatos estandarizados
- **Mantenimiento**: Se actualizan y mejoran constantemente sin necesidad de intervención manual
- **Multilingüe**: Funcionan correctamente en diferentes idiomas y contextos culturales

:::tip Consejo
Utiliza las hojas del sistema siempre que sea posible en lugar de crear hojas personalizadas para estos tipos de datos. Esto mejorará la precisión de extracción y reducirá el tiempo de desarrollo.
:::
