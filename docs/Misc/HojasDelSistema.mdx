# Hojas del Sistema

Las Hojas del Sistema en TreeFlow son entidades predefinidas que están optimizadas para reconocer y extraer tipos de datos comunes sin necesidad de configuración adicional. Estas hojas facilitan la creación de experiencias conversacionales robustas al proporcionar extractores de información ya entrenados y listos para usar.

![Hojas del Sistema](/img/docs/hojas_sistema.svg)

## Lista Completa de Hojas del Sistema

| Hoja | Descripción | Ejemplos de formato |
|------|-------------|--------------------| 
| **sys.any** | Captura cualquier texto o fragmento de información. Útil cuando necesitas extraer texto libre sin estructura específica. | "me gusta", "quiero información", "cualquier texto que el usuario escriba" |
| **sys.number** | Reconoce números en diversos formatos, incluyendo números escritos como texto, decimales, y expresiones numéricas. | "42", "tres", "3.14", "mil doscientos", "un millón", "docena y media" |
| **sys.currency** | Identifica valores monetarios, incluyendo el símbolo de la moneda y la cantidad. Compatible con múltiples divisas y formatos regionales. | "$100", "50 euros", "1,200.50 pesos", "medio millón de dólares" |
| **sys.date** | Extrae fechas en múltiples formatos, incluyendo fechas absolutas, relativas y expresiones temporales. | "15 de marzo", "15/03/2025", "mañana", "próximo lunes", "en dos semanas", "a finales de mes" |
| **sys.time** | Reconoce expresiones de tiempo, tanto específicas como relativas. | "3 PM", "15:30", "mediodía", "en 5 minutos", "a media tarde", "después de la cena" |
| **sys.datetime** | Combina la funcionalidad de sys.date y sys.time para capturar fechas y horas juntas. | "mañana a las 3 PM", "15 de marzo a las 15:30", "próximo lunes por la tarde" |
| **sys.email** | Identifica direcciones de correo electrónico con validación de formato. | "usuario@ejemplo.com", "nombre.apellido@dominio.co.uk" |
| **sys.phone** | Captura números de teléfono en diversos formatos regionales, con o sin códigos de país. | "+1 555-123-4567", "555.123.4567", "(01) 234 5678" |
| **sys.url** | Reconoce URLs completas o parciales, con o sin protocolo. | "https://ejemplo.com", "ejemplo.com/pagina", "www.sitio.net" |
| **sys.percentage** | Identifica porcentajes expresados con símbolo o en texto. | "50%", "cincuenta por ciento", "mitad", "tres cuartos" |
| **sys.age** | Reconoce expresiones de edad en años, meses o combinaciones. | "30 años", "18 meses", "año y medio", "veinticinco años" |
| **sys.duration** | Captura períodos de tiempo o duraciones. | "2 horas", "30 minutos", "una semana", "tres días y medio" |
| **sys.address** | Identifica direcciones postales completas o parciales. | "Calle Principal 123", "Avenida Central, Edificio Torre, Piso 5" |
| **sys.name** | Reconoce nombres propios de personas. | "Juan Pérez", "María", "Dr. Rodríguez" |
| **sys.color** | Captura nombres de colores y tonalidades. | "rojo", "azul marino", "verde claro", "magenta" |

## Uso Recomendado

Las Hojas del Sistema están diseñadas para facilitar la extracción de información común sin necesidad de crear hojas personalizadas. Para utilizarlas:

1. Incluye la referencia a la hoja del sistema en tus patrones de rama usando la sintaxis `@sys.nombre_hoja`
2. TreeFlow automáticamente extraerá y normalizará los valores que coincidan con el formato esperado

### Ejemplo en un Patrón de Rama

```
Quiero reservar una mesa para el @sys.date a las @sys.time
Necesito información sobre mi pedido #@sys.number
Envíame los detalles a @sys.email
```

## Ventajas de las Hojas del Sistema

- **Robustez**: Están optimizadas para reconocer múltiples variaciones y formatos regionales
- **Normalización**: Convierten automáticamente los valores extraídos a formatos estandarizados
- **Mantenimiento**: Se actualizan y mejoran constantemente sin necesidad de intervención manual
- **Multilingüe**: Funcionan correctamente en diferentes idiomas y contextos culturales

:::tip Consejo
Utiliza las hojas del sistema siempre que sea posible en lugar de crear hojas personalizadas para estos tipos de datos. Esto mejorará la precisión de extracción y reducirá el tiempo de desarrollo.
:::
