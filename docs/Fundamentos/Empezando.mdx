# Primeros Pasos con TreeFlow

:::tip Recomendación

Antes de seguir esta guía, te sugerimos revisar primero la sección [Empieza Aquí](/docs/intro) y [Terminología](/docs/Fundamentos/Terminologia) para familiarizarte con los conceptos básicos de TreeFlow y entender la estructura de Árboles, Ramas y Hojas que utilizamos en nuestra plataforma.

:::

## Crear tu Primer Árbol

:::tip Preparación previa

Antes de crear un Árbol en TreeFlow, es recomendable tener listo tu flujo conversacional. Un buen diseño conversacional facilitará enormemente la implementación y mejorará la experiencia del usuario. Consulta nuestra guía de [Diseño Conversacional](/docs/Fundamentos/DisenoConversacional) para aprender a estructurar eficazmente tus conversaciones.

:::

![Crear nuevo árbol](/img/empezando/nuevo-arbol.svg)

### Paso 1: Crear un Nuevo Árbol

1. En la página principal, haz clic en el botón "Nuevo Árbol"
2. Proporciona un nombre descriptivo para tu Árbol (ej. "Asistente de Reservas")
3. Opcionalmente, añade una descripción que explique el propósito del Árbol
4. Haz clic en "Crear" para generar tu nuevo Árbol

### Paso 2: Configurar las Propiedades Básicas

1. En la página de configuración del Árbol, establece:
   - **Propósito**: Define claramente el objetivo de tu Árbol
   - **Configuración básica**: Revisa los ajustes predeterminados

2. Guarda los cambios

### Paso 3: Configurar Hojas

Las Hojas te permiten extraer información específica de los mensajes del usuario:

1. En el menú lateral, haz clic en "Hojas"
2. Haz clic en "Nueva Hoja"
3. Selecciona el tipo de Hoja (ej. "Fecha", "Nombre", "Número")
4. Configura los parámetros específicos para ese tipo de Hoja
5. Guarda la Hoja

Repite este proceso para crear todas las Hojas que necesitarás según tu flujo conversacional.

### Paso 4: Crear tus Primeras Ramas

![Editor de ramas](/img/empezando/editor-rama.svg)

Una vez que tengas tus Hojas definidas, puedes crear las Ramas que representan los patrones de conversación que tu Árbol puede reconocer:

1. Navega a la sección "Ramas"
2. Haz clic en "Nueva Rama"
3. Define un nombre descriptivo para tu Rama (ej. "Bienvenida")
4. Agrega patrones de activación (frases que activarán esta Rama)
5. Configura la respuesta que dará el chatbot cuando se active esta Rama
6. Guarda los cambios

:::tip Captura automática de parámetros
Cuando escribas patrones de activación, si incluyes valores clave que coinciden con los previamente definidos en tus Hojas, TreeFlow los capturará automáticamente como parámetros. Esto facilita enormemente la extracción de información estructurada de las conversaciones.
:::

Repite este proceso para crear todas las Ramas necesarias para tu caso de uso.

### Paso 5: Entrenar y Probar tu Árbol

![Probando el árbol](/img/empezando/testing-arbol.svg)

Una vez configurado tu Árbol con sus Ramas y Hojas, es necesario entrenarlo antes de poder probarlo:

1. Haz clic en el botón "Entrenar" en la parte superior derecha
2. Espera a que el proceso de entrenamiento se complete (esto puede tomar unos minutos dependiendo de la complejidad de tu Árbol)
3. Una vez entrenado, haz clic en el botón "Probar"
4. Se abrirá una ventana de chat donde podrás interactuar con tu Árbol
5. Escribe mensajes para ver cómo responde
6. Verifica que las Ramas se activen correctamente
7. Comprueba que las Hojas extraigan la información esperada

:::important Entrenamiento obligatorio
Recuerda que cada vez que realices cambios significativos en las Ramas o Hojas de tu Árbol, deberás volver a entrenarlo para que estos cambios se reflejen en su comportamiento.
:::

Si encuentras algún problema, regresa al editor y ajusta tus Ramas o Hojas según sea necesario.

## Flujo de Trabajo Recomendado

Para desarrollar Árboles efectivos, recomendamos seguir este flujo de trabajo:

1. **Planificación**:
   - Define claramente el propósito del Árbol
   - Diseña el flujo conversacional completo
   - Identifica los valores clave que necesitarás extraer

2. **Creación de Hojas**:
   - Define todas las Hojas basadas en los valores clave identificados
   - Añade variaciones y sinónimos para cada valor
   - Configura las propiedades específicas de cada tipo de Hoja

3. **Estructura de Flujos y Respuestas**:
   - Diseña la estructura de tus Ramas utilizando las Hojas ya definidas
   - Configura las respuestas para cada Rama
   - Establece los flujos de conversación con contextos

4. **Pruebas y Refinamiento**:
   - Entrena tu Árbol
   - Prueba exhaustivamente todas las rutas de conversación
   - Identifica y corrige problemas

5. **Implementación**:
   - Integra tu Árbol con los canales deseados
   - Monitorea el rendimiento
   - Realiza mejoras continuas

## Mejores Prácticas para Principiantes

1. **Orden correcto**: Siempre crea primero las Hojas y luego las Ramas, ya que las Ramas dependen de las Hojas para funcionar correctamente

2. **Hojas enriquecidas**: Genera tus Hojas con abundantes variaciones y sinónimos para mejorar la precisión en la extracción de información

3. **Comienza Simple**: Crea un Árbol con pocas Ramas y ve expandiéndolo gradualmente

4. **Patrones Diversos**: Incluye múltiples formas en que los usuarios podrían expresar la misma intención

5. **Prueba Regularmente**: Entrena y verifica que tu Árbol responde correctamente en diferentes escenarios

6. **Contextos Claros**: Utiliza contextos para guiar el flujo de la conversación

7. **Respuestas Naturales**: Proporciona múltiples variaciones de respuestas para hacer la conversación más natural
