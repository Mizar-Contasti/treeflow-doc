# Primeros Pasos con Treeflow

Bienvenido a Treeflow, la plataforma para desarrollar chatbots deterministas potenciados por modelos generativos. Esta guía te ayudará a dar tus primeros pasos para crear tu primer Tree (árbol) conversacional.

## ¿Qué es Treeflow?

Treeflow es una WebApp para desarrollar chatbots deterministas enfocados en la efectividad a bajo costo. Combina el control preciso de los sistemas deterministas con la flexibilidad de los modelos generativos, permitiéndote aprovechar las ventajas de ambos mundos.

### Conceptos Clave

Antes de comenzar, es importante familiarizarse con la terminología específica de Treeflow:

- **Trees (Árboles)**: Son los chatbots completos, equivalentes a los "bots" en otras plataformas
- **Branches (Ramas)**: Son las intenciones o patrones de conversación que tu Tree puede reconocer
- **Leaves (Hojas)**: Son las entidades o tipos de datos que tu Tree puede extraer de los mensajes

## Requisitos Previos

Para comenzar a utilizar Treeflow, necesitarás:

- Navegador web moderno (Chrome, Firefox, Edge o Safari)
- Conocimientos básicos sobre chatbots y flujos conversacionales

Una vez dentro, verás el dashboard principal con tus Trees existentes o la opción para crear uno nuevo.

## Crear tu Primer Tree

### Paso 1: Crear un Nuevo Tree

1. En la página principal, haz clic en el botón "Nuevo Tree"
2. Proporciona un nombre descriptivo para tu Tree (ej. "Asistente de Reservas")
3. Opcionalmente, añade una descripción que explique el propósito del Tree
4. Haz clic en "Crear" para generar tu nuevo Tree

### Paso 2: Configurar las Propiedades Básicas

1. En la página de configuración del Tree, establece:
   - **Propósito**: Define claramente el objetivo de tu Tree
   - **Umbral de confianza ML**: Recomendamos comenzar con 0.7
   - **Umbral de confianza Fuzzy**: Recomendamos comenzar con 0.6

2. Guarda los cambios

### Paso 3: Crear tus Primeras Branches

1. Navega a la sección "Branches" (Ramas)
2. Haz clic en "Nueva Branch"
3. Selecciona "Conversacional" como tipo
4. Proporciona un nombre descriptivo (ej. "Saludo")
5. Añade patrones de ejemplo:
   - "Hola"
   - "Buenos días"
   - "Saludos"
6. Añade respuestas:
   - "¡Hola! ¿En qué puedo ayudarte hoy?"
   - "¡Bienvenido! Estoy aquí para asistirte."
7. Guarda la Branch

### Paso 4: Crear una Branch con Parámetros

1. Crea una nueva Branch llamada "Consulta_Producto"
2. Añade patrones como:
   - "Quiero información sobre @producto"
   - "Háblame de @producto"
3. Crea un parámetro:
   - **Nombre**: producto
   - **Requerido**: Sí
   - **Prompt**: "¿Sobre qué producto necesitas información?"
4. Añade respuestas:
   - "Aquí tienes información sobre {'{'}producto{'}'}..."
5. Guarda la Branch

### Paso 5: Crear tu Primera Leaf

1. Navega a la sección "Leaves" (Hojas)
2. Haz clic en "Nueva Leaf"
3. Selecciona "Simple" como tipo
4. Proporciona un nombre (ej. "producto")
5. Añade valores y sinónimos:
   - Valor: "laptop"
     - Sinónimos: "portátil", "notebook", "computadora portátil"
   - Valor: "smartphone"
     - Sinónimos: "celular", "teléfono", "móvil"
6. Guarda la Leaf

### Paso 6: Probar tu Tree

1. Navega a la sección "Conversaciones"
2. Utiliza el panel de chat para interactuar con tu Tree
3. Prueba diferentes entradas:
   - "Hola"
   - "Quiero información sobre laptops"
   - "Háblame de smartphones"

## Flujo de Trabajo Recomendado

Para desarrollar Trees efectivos, recomendamos seguir este flujo de trabajo:

1. **Planificación**:
   - Define claramente el propósito del Tree
   - Identifica los principales flujos de conversación
   - Determina qué información necesitas extraer

2. **Estructura Básica**:
   - Crea las Branches principales
   - Configura las respuestas básicas
   - Establece los flujos de conversación con contextos

3. **Enriquecimiento**:
   - Añade más patrones a tus Branches
   - Crea Leaves para extraer información
   - Implementa respuestas más detalladas

4. **Pruebas y Refinamiento**:
   - Prueba exhaustivamente tu Tree
   - Identifica y corrige problemas
   - Añade más variaciones y mejoras

5. **Implementación**:
   - Integra tu Tree con los canales deseados
   - Monitorea el rendimiento
   - Realiza mejoras continuas

## Mejores Prácticas para Principiantes

1. **Comienza Simple**: Crea un Tree con pocas Branches y ve expandiéndolo gradualmente
2. **Patrones Diversos**: Incluye múltiples formas en que los usuarios podrían expresar la misma intención
3. **Prueba Regularmente**: Verifica que tu Tree responde correctamente en diferentes escenarios
4. **Contextos Claros**: Utiliza contextos para guiar el flujo de la conversación
5. **Respuestas Naturales**: Proporciona múltiples variaciones de respuestas para hacer la conversación más natural
